Sign in with Telegram

Authenticate via Telegram OpenID Connect. A signed id_token is verified server-side; no passwords involved.

  1. Click the button — a Telegram popup opens.
  2. Approve the login inside Telegram.
  3. Server validates the JWT against Telegram's JWKS.
  4. Cookie session is set; you're redirected to /app.
